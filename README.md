# FoneFixoSearch
Procure por todas as informações públicas sobre uma linha telefonica Brasileira

## Como utilizar?
É recomendado ter o Python 3.9 instalado. (Pode funcionar em algumas versões do 3.8)

Clone o repositório: ```$ git clone https://github.com/natan-fernandes/FoneFixoSearch```

Instale os requisitos na pasta do clone: `python -m pip install -r requirements.txt`

### Argumentos
Argumento | Descrição
------------ | -------------
-n ou -nome | Parte do nome do dono da linha
-t ou -telefone | Parte do numero do telefone
-c ou -cidade | Parte do nome da cidade

#### Exemplo
`python ffs.py -n joao das couves -c sao paulo`

`python ffs.py -n joao maria -c rio de janeiro`

`python ffs.py -c sao paulo -t 1132137`

`python ffs.py -t 3213213`
